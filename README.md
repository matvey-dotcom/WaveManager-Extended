# WaveManager Extended
Добавляет пункты 150 и 200 в выпадающий список истории треков на [WaveManager](https://core.waveradio.org/public/tool).
## В чём суть?
Официальный инструмент позволяет просмотреть лишь 10, 20, 50 или 100 последних треков. **WaveManager Extended** расширяет это ограничение до 200.
## Как работает?
После установки скрипта в интерфейсе WaveManager появятся две новые опции. Работает для всех станций:
- [Provodach](https://provoda.ch/)
- [Soviet](https://sovietwave.su/)
- [Witch](https://witch.waveradio.org/)
- [Nyanserver](https://radio.nyan.pw/)

# Установка
1. Перейдите на сайт **[Tampermonkey](https://www.tampermonkey.net/)**.
2. Выберите свой браузер и установите расширение.
3. Скачайте файл script.js из этого репозитория.
4. Импортируйте в Tampermonkey:
- Нажмите на иконку Tampermonkey в браузере
- Выберите «Панель управления»
- Перейдите во вкладку «Утилиты»
- Выберите пункт «Импорт из файла»
- Выберите скачанный файл `script.js`
- Нажмите на кнопку «Установить»
